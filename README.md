# gesture

A image gesture detection Qt application

# Environment

- Windows 10 x64
- Qt5.15.2
- OpenCV 4.8.0
- ONNX Runtime 1.12.0
- VC16 or MinGW 8.10

# feature

- use for yolov5 export onnx detection model

- surported picture and video (.mp4, .avi)

- gesture characteristic:

| class | name |
|:-----:|:----:|
| 0     | 拳    |
| 1     | 一    |
| 2     | 二    |
| 3     | 三    |
| 4     | 五    |
