# gesture

A image hand gesture number detection of Qt application

# Environment

- Windows 10 x64
- Qt5.15.2
- OpenCV 4.8.0
- ONNX Runtime 1.12.0
- VC16 or MinGW 8.10

# feature

- use for yolov5 export onnx detection model

- surported picture and video (.mp4, .avi)

- gesture model characteristic:

| class | name  |
|:-----:|:-----:|
| 0     | fist  |
| 1     | one   |
| 2     | two   |
| 3     | three |
| 4     | five  |
| 5     | four  |
